# Student Management System

A Django-based Student Management System developed using Python and Django.

## Features

- Add Students
- View Students
- Update Student Details
- Delete Students
- Search Students
- Django Admin Panel
- Bootstrap Responsive UI

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

```bash
git clone https://github.com/Remiel199/student-management-system.git

cd student-management-system

python manage.py migrate

python manage.py runserver
